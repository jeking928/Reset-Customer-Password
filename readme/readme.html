PURPOSE & AIM

This plugin allows you to reset a customer password from the Admin. You can search for a customer by their email address.

INSTALLATION

!!!!!BACKUP YOUR STORE BEFORE YOU BEGIN!!!!!

New Installation

  1. Select the appropriate folder for your version of Zen Cart.
  2. Rename YOUR-ADMIN to match your store admin directory name.
  3. Upload all files. There are no core file overwrites.
  4. If you are using Zen Cart 1.5.x run sql 'add_admin_menu_entry_v15x.sql'.
    If you are using a table prefix, you MUST EDIT the sql statement or you will get an error. For example, if your table prefix is 'zen_' change 'admin_pages' to 'zen_admin_pages'.

Upgrade Installation

  1. Replace all files using the correct folder based upon your version of Zen Cart.
  2. There is no need to re-run the sql file if you are using Zen Cart 1.5.x

Usage

  1. Under Admin>Tools, select 'Change Customer Password'.
  2. In the 'Search email' field, enter the email address of the customer for which you want to change the password.
  3. In the right column, click the big red 'CHANGE PASSWORD'.
  4. Enter the new password in the ' New Password' box and click update.
